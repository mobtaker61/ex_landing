===========================================
Virtual Tour Upload Guide
===========================================

To add a new virtual tour project:

1. Create a new folder in this directory (tours)
2. The folder name will be used as the tour name
3. Place all virtual tour files in this folder
4. Make sure index.html or index.htm exists in the folder
5. (Optional) Add a thumbnail.png file for the preview image

Example structure:
tours/
  └── my-tour-project/
      ├── index.html
      ├── thumbnail.png
      ├── script.js
      ├── styles.css
      └── ... other project files

After uploading, the tour will automatically appear in the list.
