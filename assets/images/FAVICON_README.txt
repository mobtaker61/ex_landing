===========================================
Favicon Files Guide
===========================================

Please upload the following favicon files to this directory (assets/images/):

1. favicon.ico (16x16 or 32x32 pixels)
   - Main favicon file
   - Format: ICO
   - Recommended size: 32x32 or 16x16

2. favicon-32x32.png (optional but recommended)
   - For modern browsers
   - Format: PNG
   - Size: 32x32 pixels

3. favicon-16x16.png (optional but recommended)
   - For modern browsers
   - Format: PNG
   - Size: 16x16 pixels

4. apple-touch-icon.png (optional but recommended)
   - For iOS devices (home screen icon)
   - Format: PNG
   - Size: 180x180 pixels

Note: If you only have one favicon file, name it "favicon.ico" and place it in this directory.
The other sizes are optional but improve compatibility across different devices and browsers.

You can use online tools to generate all sizes from a single image:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
