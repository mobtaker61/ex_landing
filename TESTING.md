# راهنمای تست پروژه

## روش 1: استفاده از PHP Built-in Server (سریع‌ترین روش)

### مرحله 1: باز کردن Terminal/PowerShell
در مسیر پروژه خود باشید:
```bash
cd "d:\Site Developing\Tour_RoniPlus"
```

### مرحله 2: اجرای سرور PHP
```bash
php -S localhost:8000
```

### مرحله 3: باز کردن مرورگر
مرورگر خود را باز کنید و به آدرس زیر بروید:
```
http://localhost:8000
```

### توقف سرور
برای توقف سرور، در ترمینال `Ctrl + C` را فشار دهید.

---

## روش 2: استفاده از XAMPP/WAMP/MAMP

### اگر XAMPP دارید:

1. **کپی کردن پروژه:**
   - پروژه را در پوشه `htdocs` کپی کنید:
   ```
   C:\xampp\htdocs\Tour_RoniPlus
   ```

2. **اجرای Apache:**
   - XAMPP Control Panel را باز کنید
   - Apache را Start کنید

3. **باز کردن مرورگر:**
   ```
   http://localhost/Tour_RoniPlus
   ```

## روش 3: آپلود به cPanel (تست روی سرور واقعی)

### مرحله 1: فشرده‌سازی پروژه
- تمام فایل‌ها را به جز `node_modules` (اگر وجود دارد) فشرده کنید

### مرحله 2: آپلود به cPanel
1. وارد cPanel شوید
2. به File Manager بروید
3. فایل فشرده را آپلود کنید
4. Extract کنید

### مرحله 3: تنظیم مجوزها
- پوشه‌های `tours` و `media` را 755 کنید

### مرحله 4: تست
- آدرس سایت خود را در مرورگر باز کنید

## رفع مشکلات رایج

### مشکل: صفحه سفید نمایش داده می‌شود
**راه حل:**
- بررسی کنید که PHP نصب است: `php --version`
- بررسی کنید که فایل `includes/functions.php` وجود دارد
- خطاهای PHP را در ترمینال بررسی کنید

### مشکل: تصاویر لود نمی‌شوند
**راه حل:**
- بررسی کنید که فایل‌های `thumbnail.png` در پوشه‌های پروژه وجود دارند
- بررسی کنید که مسیر فایل‌ها درست است

### مشکل: پروژه‌ها نمایش داده نمی‌شوند
**راه حل:**
- بررسی کنید که پوشه‌های `tours` و `media` وجود دارند
- بررسی کنید که در هر پوشه پروژه، فایل `index.html` یا `index.htm` وجود دارد (برای تورها)
- بررسی کنید که در هر پوشه مدیا، فایل ویدیو وجود دارد

### مشکل: خطای 404 برای فایل‌ها
**راه حل:**
- بررسی کنید که `.htaccess` درست کار می‌کند
- اگر از PHP built-in server استفاده می‌کنید، `.htaccess` ممکن است کار نکند (این طبیعی است)

---

## نکات مهم

1. **PHP Built-in Server:** این سرور فقط برای تست است و نباید در production استفاده شود
2. **مسیرها:** مطمئن شوید که مسیرهای فایل‌ها درست هستند
3. **مجوزها:** در سرور واقعی، مجوزهای پوشه‌ها را بررسی کنید
4. **خطاها:** همیشه خطاهای PHP را در ترمینال یا لاگ‌ها بررسی کنید

## تست با پروژه‌های نمونه

برای تست کامل، می‌توانید:

1. یک پوشه نمونه در `tours` ایجاد کنید:
   ```
   tours/sample-tour/
   ├── index.html (یک فایل HTML ساده)
   └── thumbnail.png (یک تصویر)
   ```

2. یک پوشه نمونه در `media` ایجاد کنید:
   ```
   media/sample-media/
   ├── video.mp4 (یک فایل ویدیو)
   └── thumbnail.png (یک تصویر)
   ```

سپس بررسی کنید که این پروژه‌ها در لیست نمایش داده می‌شوند.
