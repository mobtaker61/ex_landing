===========================================
Exhibitions List File Guide
===========================================

This file (exhibitions.txt) contains the list of exhibitions that appear in the request form dropdown.

HOW TO EDIT:
------------
1. Open exhibitions.txt file directly on your server
2. Add one exhibition name per line
3. Save the file
4. The changes will appear immediately in the form dropdown

FORMAT:
-------
- One exhibition name per line
- No special characters needed
- Empty lines will be ignored
- Maximum recommended: 50 exhibitions

EXAMPLE:
--------
Dubai International Exhibition Centre
Dubai World Trade Centre
Dubai Festival City
Abu Dhabi National Exhibition Centre

IMPORTANT NOTES:
----------------
- This file is NOT tracked in Git (for security)
- Always keep a backup of this file
- Use exhibitions.txt.example as a template if file is deleted
- File encoding should be UTF-8
